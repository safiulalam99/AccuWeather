# Weather app


## Goals
Recieve weather data from [Accuweather.com](developer.accuweather.com) API and format them on a card with styling.

## App Description
- Submit form for location query.
- Displays realtime weather data.

## Tech Stack
 - [React.js](https://reactjs.org/)
 - Axios
 - Hooks
